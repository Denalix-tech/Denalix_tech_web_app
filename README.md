# Denalix Tech Web App

Modern Next.js website for Denalix Tech, an AI and software consulting company.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- shadcn-style UI primitives
- Framer Motion
- Lucide React icons

## Notes

The contact form currently uses a placeholder submission handler in `components/ContactForm.tsx`. It is structured so it can later connect to Supabase, Resend, HubSpot, Zapier, or a CRM endpoint.
